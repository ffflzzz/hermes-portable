【使用方法】

1. 双击 "hermes.bat" (Windows) 或 "hermes.sh" (Linux/Mac)

2. 首次运行会提示输入 API Key
   - 如果你已经买了API Key，直接粘贴进去
   - API Key会自动保存，下次不用重复输入

3. 输入问题，回车即可
   例如：
   - "我的电脑开不了机怎么办？"
   - "蓝屏了怎么修复？"
   - "文件找不到了"

4. 常用命令：
   - /clear    清空对话
   - /reset    重置API Key
   - /info     查看系统信息
   - /quit     退出


【注意事项】

- 需要联网才能使用
- API Key在你购买时获得
- 所有对话记录保存在 .hermes_portable 文件夹
- 本工具免安装，放在U盘里随时可用
